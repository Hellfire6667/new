Умный унитаз

Когда люди покупают умные вещи, то думают, что они умные во всем, но к счастью создателя ботнета mirai большинство людей не думает об этом и оставляют на устройствах стандартные порты и комбинации логина и пароля. Например, при покупке умного унитаза, вам первым делом необходимо настроить его, через веб-интерфейс или мобильное приложение, ведь если вы не настроите свой унитаз, то вам нужно будет надеятся, что унитаз определяет ваш профиль по весу, а не по фото.

Сканирование портов

Как я говорил на примере унитаза-очень важно менять и защищать порты, но сейчас мы посмотрим как посмотреть эти порты.

Для сканирования TCP портов существует простая утилитп hping3, она идет в стандартном пакете кали линукс, пример команды: hping3 -c 1 -V -p 80 -s 5050 -A example.com

Но кому нужны эти утилиты, когда есть утилиты побольше, такие как:

NMAP

Это один из лучших сканеров портов, здесь будет представлена краткая информация по сканированию

nmap 192.168.1.*

Сканирование диапазона адресов

nmap 192.168.1.0/24

Всей подсети

nmap -A 192.168.1.254
nmap -v -A 192.168.1.1

Определение ОС и ее версии на пк жертвы

nmap -sA 192.168.1.254

Проверка на наличие фаервола

nmap -PN 192.168.1.1

Сканирование с обходом фаервола

nmap -sP 192.168.1.0/24

Поиск включеных устройств

nmap --open 192.168.1.1

Поиск открытых портов

nmap --iflist

Поиск интерфейсов(помним про унитаз)

Если вы фанат окошек, то вы можете использовать графическую версию данной программы zenmap

Striker

разведка для высокооптимизированных(ленивых)

Написаный на питоне инструмент автоматического сканирования и сбора информации из гугла, скачать можно с гитхаба

git clone https://github.com/s0md3v/Striker.git

Также вам необходимо установить pip, перейти в директорию, где лежит наш файл и вводим следующие команды

pip install –r requirements.txt
pip install whois
pip install who is

даже если уже установили

python striker.py

Данная программа платная, но нам хватит и обрезаной версии. Необходимо указать свою почту на сайте программы ТуТ

Для работы с утилитой перейдите в браузере по адресу https://localhost:8834

OpenVAS

Для установки введите

apt-get install openvas
openvas-setup

После этого сгенирируется пароль, необходимый для входа. Для последующего запуска введите

Зайдите в браузер и введите https://localhost:9392

Данная утилита похожа на Нессус, поэтому сложностей возникнуть не должно

Lynis

Данная программа входит в стандартный пакет кали линукс. Для запуска введите в командную строку

lynis

Для просмотра команд введите lynis show commands

Чтобы проверить свою машину, введите

Проверка длиться от 15 минут, зависит от машины

SPARTA

Данная утилита есть в пакете кали линукс, также можно скачать ее с гитхаба git clone https://github.com/secforce/sparta.git После запуска спарты вам предложат ввести диапазон айпи адресов, нажмите Add to scope, после этого начнется пошаговое сканирование. После этого повятся вкладки с информацией об уязвимостях, если рядом с отчетом написано OSVBD, значит что существует эксплойт для данной уязвимости на сайте ТЫК и ТУТ

Настройка программ

Некоторые сканирующие программы могут выдать ваш айпи, поэтому необходимо сделать следующие вещи

sudo apt-get install torsocks tor

Добавьте следующие строки в конец файла /etc/tor/torrc

AutomapHostsOnResolve 1
DNSPort 53530
TransPort 9040

Добавьте Тор в автозапуск

sudo systemctl start tor
sudo systemctl enable tor

ProxyChains-NG

Прокси чейн обеспечивает дополнительную анонимность

Установка

sudo apt-get remove proxychains
git clone https://github.com/rofl0r/proxychains-ng.git
cd proxychains-ng/
./configure --prefix=/usr --sysconfdir=/etc
make
sudo make install
sudo make install-config

Сканирование NMAP через прокси

Для этого добавьте флаг -st

-Pn (Не использовать пинг сканирование)
-sV (Определение версии)
--open (Показывать только открытые (или возможно открытые) порты)

Для сканирование через прокси используется подобный синтаксис

proxychains4 nmap -sT -PN -sV --open -n -p порт адрес

Если у вас возникает ошибка при сканировании, откройте файл /etc/proxychains.conf и закоментируйте строку

proxy_dns
должно получится вот так #proxy_dns

используется все это вот так

Сканирование WPScan через TOR

используется все это вот так

wpscan -u "ЦЕЛЬ" -e p,vt,u --proxy socks5://127.0.0.1:9050

RAT

Ремоте администрейшон тулс.

Для этого мы используем Social Engineering Toolkit. Для запуска введите в командную строку setoolkit

Создание зараженного exe файла RAT

Для этого выберите пункт Social Engineering Attacks. Dам будет предложена множество вариантов атак, выберите Infectious Media Generator для создания зараженного исполняемого файла. После этого мы выбираем Standard Metasploit Executable. Нужно будет выбрать вариант соединения, в коорпоративных сетях удобно использовать Meterspeter Reverse HTTPS, для простого подключения к компьютеру мы выбираем ) Windows Reverse_TCP Meterpreter . Далее вам нужно ввести свой айпи и порт, например 4444. Будет предложено создать прослушиватель, вы соглашаетесь. Сгенерированный файл лежит в /root/.set, после этого вам нужно отправить этот файл по почте или перенести на флешке на машину жертвы.

Создаем фишинговый вебсайт

Для этого той же программе выбираем Website Attack Vectors, выбираем метод Credential Harvester, для сбора данных. Создаем клон сайта, выбираем Site Cloner, после этого вам нужно указать айпи сайта, можете указать свой. Выбираем сайт, который будем копировать, в нашем случае https://vk.com/login после этого переходим по своему айпи в браузере для проверки, отправляем эту ссылку в виде гиперссылке нашей жертве.

Создаем Java-аплет

Для этого опять же заходим в setoolkit, выбираем Website Attack Vectors, после выбираем Java Applet Attack, выбираем клонирование сайта, вас спросят используете ли вы NAT, если вы находитесь в одной сети с жертвой, введите no, программа автоматически узнает ваш айпи адрес, вам нужно лишь нажать на ВВОД, после выбираем Use the applet built into SET, если знаете язык джава, выбираете другой вариант. Вам предложат ввести адрес сайта, после этого вам предложат выбрать полезную нагрузку, в нашем случае SE Toolkit Interactive shell designed for SET. Необходимо будет выбрать порт для прослушки, по умолчанию 443. После этого отправляем адрес нашего сайта жертве и ждем.
